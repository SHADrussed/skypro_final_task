Git

  • git init – создать новый локальный репозиторий в текущей папке.
  • git clone url – скопировать удалённый репозиторий на свой компьютер.
  • git status – показать состояние файлов (изменённые, добавленные, неотслеживаемые).
  • git add файл – добавить файл в индекс (подготовить к коммиту).
  • git log – показать историю коммитов (с хешами, авторами, датами).
